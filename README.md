Quizmoney
---------
This is a gaming web app. Quiz Money Game is an online quiz game where players are asked common knowledge by playing quiz questions and answering these questions correctly. Players who answer a certain number of questions correctly get a chance to spin the wheel and win cash prices. Cash prices can be redeemed once a player gets N1000 and above and follows the basic steps in order to redeem cash prices. This step is described on our website.

Tech Stacks Used
-----------------
HTML, CSS, BootStrap, JavaScript, JQuery, PHP and MySQL
